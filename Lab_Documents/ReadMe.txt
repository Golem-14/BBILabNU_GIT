This folder is dedicated for storage of various laboratory documents:

- Protocols 
- SOPs, etc.

The files should be presented in the Markdown format to ensure that their modifications can be traced in the same way the changes in the scripts are traced.

Each group or project can request a separate folder or use the general one provided proper naming and commenting of the corresponding documents to avoid any confusion.

As example there is a protocol for functionalization. You can use this file as a template for your own contributions.
